Array.prototype.some()

A method that tests whether any one element in a collection satisfies a given condition.

Check whether atleast 1 person is 19.

                const isAdult = people.some(function(person) {
                    const currentYear = (new Date()).getFullYear()
                    if(currentYear - person.year >= 19) {
                        return true
                    }
                })

Array.prototype.every()

The every() method tests whether all elements in the array pass the test implemented by the provided function.

Check whether everyone is 19 or older.

                const allAdult = people.every(function(person) {
                    const currentYear = (new Date()).getFullYear()
                    return currentYear - person.year >= 19
                })

Array.prototype.find()

The find() method returns the first element in the provided array that satisfies the provided testing function.

Find the comment with the ID of 823423.

                const comment = comments.find(comment => comments.id === 823423)

Array.prototype.findIndex()

The findIndex() method returns the index (position) of the first element that passes a test.

Find and delete the comment with ID of 823423.

                const index = comments.findIndex(index => index.id === 823423)

                comments.splice(index, 1)
